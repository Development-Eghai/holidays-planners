==================================================================================
TROUBLESHOOTING
==================================================================================

Issue: Crawler sees empty HTML (no schema, meta tags)
  Cause: Server not using SSR middleware
  Fix:
    1. npm install express (if missing)
    2. npm run build:ssr
    3. npm run serve:ssr
    4. Test: curl -A Googlebot http://localhost:3000

Issue: Sitemap is empty or "404"
  Cause: generate-sitemap.js failed silently (API connection issue)
  Fix:
    1. Check API_KEY in scripts/generate-sitemap.js
    2. Test curl: curl -H "x-api-key: [KEY]" https://api.yaadigo.com/secure/api/trips/
    3. Ensure FastAPI is running and reachable
    4. Run: npm run sitemap (for verbose output)

Issue: Schema validation fails (Rich Results Test)
  Cause: Missing @context or invalid JSON-LD
  Fix:
    1. Open browser DevTools > Elements
    2. Find <script type="application/ld+json">
    3. Check for JSON syntax errors
    4. Use validator.schema.org to debug

Issue:  Images not showing alt text
  Cause: Components haven't been updated
  Fix:
    1. Search for <img without alt or with alt=""
    2. Add descriptive alt using format: "[Context] [Details] | Holidays Planners"
    3. Test: Disable images in browser > look for alt text
